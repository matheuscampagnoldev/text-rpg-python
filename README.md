# 🏰 Text RPG Python

Um RPG de texto desenvolvido em Python para praticar programação, lógica, estruturas de dados e orientação a objetos.

## 🚀 Objetivos

- Aprender Python na prática
- Aplicar conceitos de POO
- Trabalhar com Git e GitHub
- Construir um projeto completo do zero

## 📋 Funcionalidades Planejadas

- [ ] Sistema de combate
- [ ] Classes de personagem
- [ ] Sistema de níveis
- [ ] Inventário
- [ ] Loja
- [ ] Missões
- [ ] Bosses
- [ ] Save em JSON

## 🛠 Tecnologias

- Python
- Git
- GitHub

## 📂 Estrutura do Projeto

text-rpg/
│
├── main.py            # Ponto de entrada do jogo
├── game.py            # Loop principal do jogo
│
├── core/              # Motor do jogo (engine, batalha, utilidades)
├── characters/        # Jogador e inimigos
├── items/             # Sistema de inventário e itens
├── world/             # Mundo, missões e loja
├── data/              # Saves e dados do jogo (JSON)
└── assets/            # Lore, textos e história
