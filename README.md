# 🏰 Text RPG Python

Um RPG de texto desenvolvido em Python para praticar programação, lógica, estruturas de dados e orientação a objetos.

## 🚀 Objetivos

- Aprender Python na prática
- Aplicar conceitos de POO
- Trabalhar com Git e GitHub
- Construir um projeto completo do zero

## 📋 Funcionalidades Planejadas

- [ ] Sistema de combate
- [ ] Classes de personagem
- [ ] Sistema de níveis
- [ ] Inventário
- [ ] Loja
- [ ] Missões
- [ ] Bosses
- [ ] Save em JSON

## 🛠 Tecnologias

- Python
- Git
- GitHub

## 📂 Estrutura do Projeto

- main.py → Entrada do jogo
- game.py → Loop principal
- core/ → Engine do jogo (batalha, utilidades)
- characters/ → Jogador e inimigos
- items/ → Inventário e itens
- world/ → Missões e loja
- data/ → Saves em JSON
- assets/ → Lore e história
